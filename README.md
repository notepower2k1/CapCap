# CapCap

CapCap is a local desktop tool for creating Vietnamese subtitles and Vietnamese voice tracks for videos, with timeline-based review, live preview, exact frame preview, and a simpler guided UI for first-time users.

## Main Features

- Guided `START HERE` flow for the fastest path: choose a video, choose an output mode, and run the pipeline.
- Full subtitle workflow: audio extraction, transcription, translation, subtitle editing, preview, and export.
- Full voice workflow: generate Vietnamese voice, mix with background audio, or use an existing mixed audio file.
- Live preview tools:
  - subtitle overlay inside the player
  - auto-refresh exact frame preview while editing subtitle text/style
  - auto-refresh exact frame preview when seeking on the timeline
  - large frame preview and 5-second preview clip when needed
- Subtitle style controls:
  - font
  - size
  - color
  - positions: `Bottom Left`, `Bottom Center`, `Bottom Right`, `Center`, `Top Center`
  - horizontal and vertical offsets
- Collapsible `ADVANCED` section so the main UI stays focused for new users.
- Last-used settings persistence through `QSettings`.
- Automatic cleanup of temporary preview files, keeping only the newest preview artifacts.

## Requirements

- Windows 10/11
- Python 3.9+
- FFmpeg / FFprobe binaries in `bin/`
- Whisper binaries in `bin/whisper/`
- Whisper model files in `models/`

## Installation

1. Clone the repository:

```bash
git clone https://github.com/notepower2k1/CapCap.git
cd CapCap
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create `.env` from `.env_example` and fill in the required values.

## Run

```bash
python ui/gui.py
```

## Quick Start

### Recommended flow for most users

1. Choose a video in `START HERE`.
2. Select an output mode:
   - `Vietnamese subtitles only`
   - `Vietnamese voice only`
   - `Vietnamese subtitles + voice`
3. Click `Create Vietnamese Output`.
4. Review the result with:
   - auto exact frame preview
   - timeline seeking
   - `Open Large Frame Preview`
   - `Open 5-Second Preview`
5. If needed, edit subtitle text in the `Subtitles` section and click `Apply Edited Subtitle To Preview`.
6. Click `Export Final Video`.

### If you want to use your own audio

- `Use generated Vietnamese voice`: preview/export uses the voice or mix generated by CapCap.
- `Use existing mixed audio`: preview/export uses the external mixed audio file you selected.
- If `Background audio` is provided, CapCap mixes it into the generated voice when possible.

## UI Layout

- `START HERE`: guided primary workflow
- `Live preview`: video player, timeline, exact frame preview thumbnail
- `ADVANCED`: collapsible manual controls with 4 tabs:
  - `1. Prepare`
  - `2. Subtitles`
  - `3. Voice`
  - `4. Tools`

## Notes

- Exact frame preview uses debounced rendering to avoid excessive processing.
- Temporary preview files are cleaned up so only the newest preview is kept.
- After a successful export, temporary preview files are removed automatically.

## Project Areas

- `app/`: processing logic for audio, subtitles, preview, export, and translation
- `ui/gui.py`: main PySide6 application UI
- `bin/`: ffmpeg, ffprobe, whisper-cli binaries
- `models/`: Whisper model files
- `temp/`: temporary files
- `output/`: exported results
